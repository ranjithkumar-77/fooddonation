# 🍱 Food Donation System

## 📌 Project Overview
The Food Donation System is a Spring Boot web application that connects food donors with NGOs to reduce food waste and help people in need.

## ✨ Features
- Donor Registration & Login
- NGO Registration & Login
- Donate Food
- View My Donations
- Donation Status Tracking
- NGO Dashboard
- Accept Donations
- Schedule Pickup
- Complete Donations
- Donation History
- Google Map Location

## 🛠 Technologies Used
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap 5
- HTML
- CSS
- Git & GitHub

## 🚀 How to Run
1. Clone the repository.
2. Open the project in VS Code or Eclipse.
3. Configure MySQL in `application.properties`.
4. Run the Spring Boot application.
5. Open:
   http://localhost:8080
